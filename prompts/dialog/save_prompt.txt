# Save Game Decision Handling

You are managing save game decisions. Determine optimal save timing and frequency.

# Context Injection
{joke_state}
{recent_outcomes}
{active_goals}
{memory_context}

# Task Description
Handle save decisions:
- Evaluate save necessity
- Choose save locations
- Manage save file slots
- Handle auto-save situations
- Plan save strategy for long sessions
- Recover from bad save states

# Constraints
- Some games have limited save slots
- Save timing affects progress
- Some events save automatically
- Some bugs can corrupt saves
- Save states in emulators are different

```json
}
```

# Multi-Step Route Planning

You are planning multi-step routes for efficient Pokemon travel. Create optimal paths considering objectives and obstacles.

# Context Injection
{joke_state}
{recent_outcomes}

{memory_context}

# Task Description
Plan efficient multi-step routes:
- Identify current location and destination
- Calculate optimal path through terrain
- Consider encounter rates and danger levels
- Plan rest/recovery stops along route
- Account for HM requirements and obstacles
- Factor in time and resource constraints

# Constraints
- Some areas require specific HM moves
- Encounter rates vary by grass type and time
- Danger levels can change with team strength
- Some routes have one-way paths
- Time of day affects available routes

```json
}
```

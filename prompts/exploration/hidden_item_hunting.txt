# Task Description
Find hidden items efficiently:
- Identify areas with high hidden item density
- Use methods to reveal hidden items
- Check different terrain types
- Return to areas after story progress
- Use NPCs and hints for item locations
- Document discovered items

# Constraints
- Hidden items have specific spawn locations
- Some items appear only once
- Some require HM access to reach
- Berries replant and regrow
- Some items are time-limited

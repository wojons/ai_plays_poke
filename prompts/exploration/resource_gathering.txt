# Resource Collection Route Planning

You are planning routes to collect resources efficiently. Maximize item and money collection.

# Task Description
Plan efficient resource collection:
- Identify high-value resource locations
- Plan routes that maximize item collection
- Consider encounter efficiency (money vs items)
- Factor in shop locations for selling
- Use pick-up abilities for bonus items
- Time collection with game events

# Constraints
- Item locations are finite
- Encounters take time
- Some items require specific conditions
- Storage limits apply
- Some resources are game-time locked

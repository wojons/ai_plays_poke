# Pokedex Completion Strategy

You are planning Pokedex completion. Create systematic approach to filling the Pokedex.

# Context Injection
{joke_state}
{recent_outcomes}
{active_goals}
{memory_context}

# Task Description
Plan Pokedex completion:
- Identify current Pokedex progress
- Identify missing Pokemon by region
- Plan encounter locations for missing Pokemon
- Use methods to increase encounter chances
- Consider trading for unavailable Pokemon
- Track evolution requirements

# Constraints
- Some Pokemon are version-exclusive
- Some Pokemon require events
- Trading is needed for some evolutions
- Some areas have limited Pokemon
- Regional Pokedex vs National Pokedex
- Time investment is significant

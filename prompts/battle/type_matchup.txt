# Type Matchup Analysis

You are analyzing type matchups in a Pokemon battle. Provide detailed type effectiveness calculations and recommendations.

# Context Injection
{joke_state}
{recent_outcomes}

# Task Description
Analyze type matchups comprehensively:
- Calculate STAB (Same Type Attack Bonus) multipliers
- Identify type weaknesses and resistances
- Consider dual typing complexities
- Evaluate defensive and offensive type advantages
- Plan for enemy switch-ins and predictions
- Optimize move selection based on type coverage

# Constraints
- Know all 18 types and their interactions
- Account for abilities that modify type interactions
- Consider Field conditions that affect types
- Dual typing creates complex effectiveness charts
- Some moves have unique type interactions
